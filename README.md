# IdeaVault

Live Site URL: https://your-ideavault-site.vercel.app

IdeaVault is a web-based startup idea validation platform where users can share concepts, explore community ideas, and discuss improvements through comments.

## Features

- Public home page with a startup-focused banner slider, trending ideas, and platform stats.
- Ideas page with title search, category filter, and optional date range filter.
- Private routes for adding ideas, viewing idea details, managing personal ideas, and checking interactions.
- Authentication UI with email/password login, registration validation, Google login simulation, and persistent session storage.
- Comment system with add, edit, and delete actions for the logged-in user's own comments.
- Profile management with name and photo URL updates.
- Dark and light theme toggle that applies globally.
- Custom toast messages and confirmation modals instead of default browser alerts.
- Responsive layout for mobile, tablet, and desktop screens.
